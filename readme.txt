Апдейт зависимостей: pip freeze > requirements.txt

Установка зависимостей: pip install -r requirements.txt


Запуск проекта:

0. net start mysqlTesseract от имени администратора
1. Запустить TBot_init.py
2. Запустить Ozon_api_init.py


Команды для удобства:

mysql -u root -p (Вместо root - Samurai)
mysql -u Samurai -p
net start mysqlTesseract / net stop mysqlTesseract
